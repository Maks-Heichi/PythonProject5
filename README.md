## Курсовой проект: анализ банковских транзакций

Проект реализует бэкенд для набора страниц и сервисов, работающих с выгрузкой операций из «Т‑Банка» (Excel-файл `data/*.xlsx`).

Реализовано:

- в модуле `src/views.py`:
  - главная (`build_main_page_context`) — приветствие, суммы по картам, кешбэк, топ транзакций, курсы валют и цены акций;
  - события (`build_events_page_context`) — расходы и поступления по категориям за выбранный период;
- вспомогательные функции и работа с Excel / API в `src/utils.py`;
- сервисы поиска и анализа в `src/services.py`;
- отчёты на базе `pandas` в `src/reports.py` с сохранением результатов в файлы.

### Установка и запуск

1. Установите зависимости (через `poetry` или `pip`):

"poetry install" или "pip install -r requirements.txt"

2. Скопируйте `.env_template` в `.env` и заполните переменную:

В .env и .env_template указан мой API_KEY

3. Поместил свой Excel-файл с операциями взятыми в "Т-Банке" в папку `data` (имя `Operations.xlsx`).

4. Запустите демонстрационный модуль:

python -m src.main

5. Запустите тесты:

pytest
